URL Shortener

Your URL must be of the following format: scheme://hostname/path?query

scheme can be either http or https

hostname is the name of the website (e.g. wikipedia.org, freecodecamp.org, etc.)

path is the location of a specific resource on the website. For example, the address of the cabbage page on Wikipedia is the following: https://en.wikipedia.org/wiki/Cabbage. The path would be /wiki/Cabbage in this case. The path is not mandatory for the URL Shortener to work.

query is a string of information provided by the client and used by the resource for multiple purposes(e.g. searching). Example of query string results?search_query=yoghurt&sp=EgYIBBABGAM%253D. The query is not mandatory for the URL Shortener to work.

The response will look something like this:

{ "original_url": "https://www.wikipedia.org/wiki/Sledgehammer", "short_url": "eskeskeske"}

To verify that the short URL is working, copy and paste the value of "short_url" after the /api/short/ in the address bar(i.e. /api/short/eskeskeske)
